
# Movie Recommendation System using KNN

## Overview

This project implements a movie recommendation system using collaborative filtering and K-Nearest Neighbors (KNN).

The system analyzes user rating patterns from the MovieLens dataset and recommends movies based on user similarity.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- KNN
- Cosine Similarity

## Workflow

1. Data preprocessing
2. Exploratory Data Analysis (EDA)
3. User-item matrix creation
4. KNN model training
5. User similarity computation
6. Personalized movie recommendation generation

## Dataset

MovieLens Dataset

## Sample Output

Recommended Movies:

- Shawshank Redemption
- Fargo
- Godfather
- Godfather Part II
- Beverly Hills Cop
